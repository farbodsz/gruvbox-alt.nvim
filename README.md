# farbodsz-theme.nvim

My theme, based on [gruvbox](https://github.com/morhetz/gruvbox) colors,
written in Lua.

> **NOTE:** This is mostly intended for my own, personal use.

## Installation

Packer:

```lua 
  use("farbodsz/theme.nvim")
```

```vim
colorscheme farbodsz-theme
```
